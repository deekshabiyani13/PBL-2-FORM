Explainable AI for IoT Security using Disagreement Analysis

An AI-based intrusion detection system that detects attacks and evaluates the reliability of explanations using LIME and SHAP disagreement analysis.

Problem Statement

Traditional IoT systems cannot adapt to evolving cyber threats. Even accurate AI models lack explanation reliability, creating trust issues in security decisions.

Literature Review

Existing research focuses on improving accuracy but ignores explainability consistency.

Research Gap

No system evaluates disagreement between explainability methods like LIME and SHAP.

System Methodology

Dataset

225K+ IoT samples, 78 features, cleaned and normalized.

Model

Random Forest classifier for DDoS detection.

Execution

LIME + SHAP + Disagreement Analyzer

Results & Analysis

Accuracy 99.99%

Agreement = 0.3617 | Disagreement = 0.6383
High accuracy but high disagreement shows explanation reliability issues.
